# Profile Card project

A simple profile card built using Html and Css.This project was created while learning CSS and later rebuilt using Flexbox to improve the layout and code structure

## Features

- Semantic HTML5 structure
- CSS Flexbox layout
- Google Fonts (Poppins)
- Social media icon section
- Clean and organize code

## Technologies Used

- HTML5
- CSS3
- Flexbox

# Project Evolution 

version 1 

- Built using basic HTML and CSS.
- Focused on learning spacing, borders, and simple layouts.

version 2 

- Rebuilt using Flexbox
- Improve HTML structure
- Better aligment and spacing
- Added Social media icons
- Used Google Fonts
- Cleaner and more readble CSS

- Creating layouts more effectively
