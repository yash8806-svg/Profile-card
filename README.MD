# Profile Card project

A simple profile card built using Html and Css.

## Features

- Rounded profile Image
- Style category tags
- Centered "Read more" button
- Clean and beginner-friendly project

## Technologies Used

- Html5
- csss3

# What i learned while building this project I practiced:

- HTML structure and semantic elements
- CSS box model (Margin, Padding, border)
- Centering button using ",argin : 0 auto"
- Styling buttons and images
- Using classes and organizing css
